HTML

This webpage is made using HTML, CSS, Javascript and many othe things

Structure of a webpage

There is a DOCTYPE html tag at the top of each html file

			
<!DOCTYPE html>
<html>
 <head>
  <title></title>
 </head>
 <body>
 </body>
</html>

General Tags

				
<!DOCTYPE html>
<html>
 <head>
  <title>General Tags</title>
 </head>
 <body>
  <h1>This is a header</h1>
  <h2>This is another header</h2>
  <h3>This is a another header</h3>
  <h4>This is a another header</h4>
  <h5>This is a another header</h5>
  <h6>This is a another header</h6>
  <p>This is a paragraph</p>
 </body>
</html>

Bold, underline, itallics, strike through

				
<!DOCTYPE html>
<html>
 <head>
  <title>B.U.I.S</title>
 </head>
 <body>
  <p><strong>Bold</strong></p>
  <p><em>Italics</em></p>
  <p><u>Underlined</u></p>
  <p><strike>Striked</strike></p>
 </body>
</html>

The <strong> tag is bold. The <em> tag is itallics
The <u> tag is underline and the <strike> tag is strike through.

Divs and Spans

				
<div>
 <!--HTML goes here-->
</div>
<p>This is a <span>paragraph</span></p>

Divs and Spans will be very important when you start learning
CSS/Javascript and many others.

Note: I have also included something that looks like this: <--Text-->
That is a comment. The comment is to tell the user what each section is about.
It is only shown in the code. If you want to see the code
Press Ctrl+U for Windows, for Mac press Cmd+U.

Lists

				
<!--Unordered list-->
<ul>
 <li>Item 1</li>
 <li>Item 2</li>
 <li>Item 3</li>
 <li>Item 4</li>
</ul>
<!--Ordered list-->
<ol>
 <li>Item 1</li>
 <li>Item 2</li>
 <li>Item 3</li>
 <li>Item 4</li>
</ol>

Unordered lists look like this:

  • Item 1
  • Item 2
  • Item 3
  • Item 4

Ordered lists look like this:

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

The <li> tag is a list item. The <ul> tag is an unordered list and
the <ol> tag is a ordered list. An unordered list is a bulleted list.
You can have as much as items as you want in a list.
